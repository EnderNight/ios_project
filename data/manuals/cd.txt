
--NAME--
    cd - change the current working directory

--USAGE--
    cd [OPTION]

--OPTIONS--
    ..: Move back to the parent directory.
    <directory>: Move to the specified directory.

--DESCRIPTION--
    The cd command changes the current working directory of the shell. 
    If the .. option is specified, it moves back to the parent directory. 
    If a <directory> is specified, it moves to the specified directory.

--ARGUMENTS--
    OPTION: The option to specify the desired directory.

--SYSTEM CALLS--
    -lseak
    -getcwd
    -putchar
    -chdir
    -write
    -open
    -close
